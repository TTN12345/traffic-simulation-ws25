## Project Goal
Build a **real-time traffic simulation** using **SUMO + TraaS** with:
- 2D interactive GUI (Java Swing)
- Vehicle injection & traffic light control
- Real-time analytics
- CSV/PDF export
- Clean OOP design

---

## Tech Stack
- **Java 17+**
- **SUMO 1.18+**
- **TraaS API**
- **Java Swing** (GUI)
- **JFreeChart** (charts)
- **iText** (PDF)
- **Maven** (optional)
